# 更新说明

## 1.4.4
- **修复 Excel 导出空白文件问题**
  - 修正 Excel 写入顺序，先写入 workbook 再刷新输出流，避免系统文件选择器生成空白 xlsx 文件。
  - 导出写入器固定写入模板中的“使用说明”“测量记录”“用户资料”“导出信息”四个工作表，确保历史测量记录写入用户打开的正式工作表。
  - 表头写入改为复用现有单元格并覆盖标准列名，保持模板样式和导出列结构稳定。
  - 保留无历史记录时的诊断提示，避免生成看似成功但没有测量记录的备份文件。

## 1.4.3
- **修复 Excel 导出空白表格问题**

  - 新增保留模板标题行逻辑，确保导出格式正确。
  - 添加详细的导出诊断日志，便于问题排查。
  - 增强模板加载错误处理和提示。
